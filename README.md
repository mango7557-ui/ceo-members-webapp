# CEO Members - 프랜차이즈 플랫폼 웹앱

CEO members 프랜차이즈 플랫폼의 모바일 웹 애플리케이션입니다.

## 🌐 라이브 데모

배포된 사이트: [CEO Members](https://ceo-members.pages.dev)

## 🚀 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- Swiper.js

## 📱 주요 기능

- 모바일 최적화 반응형 디자인
- AI 프랜차이즈 Q&A
- 업체 검색 및 지도 찾기
- 카드 슬라이더 UI
- SNS 연동
- 하단 네비게이션

## 🛠️ 로컬 실행

```bash
# Python HTTP 서버
python3 -m http.server 8000

# 또는 Node.js http-server
npx http-server
```

브라우저에서 `http://localhost:8000` 접속
